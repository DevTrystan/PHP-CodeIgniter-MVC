Développer un site web en PHP, MySQL et JavaScript,
avec jQuery, CSS et HTML5
5e édition

Merci d'avoir téléchargé ces exemples

Tous les fichiers d'exemples sont présents dans les dossiers intitulés CH03 à CH27 de cette archive, en fonction de leurs chapitres correspondants. Vous devez installer un serveur web (tel que Ampps) pour exécuter la plupart de ces exemples : consultez le chapitre 2 de la 5e édition pour de plus amples détails.

En plus de la copie dans leur chapitre correspondant, les fichiers d'exemple du Nid de Robin sont rassemblés également dans le dossier nidderobin, pour que vous puissiez copier tout le dossier et ses fichiers à la racine de votre serveur web et tester l'application par vous-même. Lisez le dernier chapitre pour de plus amples détails sur la mise en place et le paramétrage de la base de données. Les autres exemples qui nécessitent des noms de fichiers distincts pour leur exécution sont aussi repris sous le dossier Exemples_nommés. Tous les autres renseignements dont vous avez besoin pour exécuter, comprendre et modifier ces exemples figurent dans le livre.

Pour les derniers détails concernant ce livre, avec notamment des errata découverts depuis la parution du livre (et les mises à jour éventuelles des fichiers), consultez le site web d'accompagnement du livre.

Conseil : Dès que votre serveur web local (Ampps ou autre) est en place, copiez la totalité de la structure des exemples avec leurs dossiers et sous-dossiers et collez-les à la racine des documents de votre serveur. Vous pourrez alors facilement naviguer parmi ces exemples en entrant localhost/CodesEtSolutions_5eEdition dans la barre d'adresse de votre navigateur.

Remarque : Le lien Nid de Robin, qui correspond au sous-dossier nidderobin, produira des erreurs si vous tentez de cliquer dessus à priori. Pour que ce lien fonctionne, vous devez d'abord créer un utilisateur et une base de données dans MySQL nommés nidderobin et exécuter le programme setup.php contenu dans ce dossier, comme l'explique le chapitre 27.
